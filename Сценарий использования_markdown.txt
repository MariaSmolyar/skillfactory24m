# Сценарий авторизации на сайте Пиццерии **Большой Папа**
1. Заходит на сайт [обычная ссылка в строке] (https://bigpapapizza.com)
2. Нажимает на иконку **Авторизация** в правом верхнем углу экрана
3. Вводит логин и пароль
4. Нажимает на кнопку **Войти**
5. После успешной авторизации просматривает личный кабинет

## Табличная часть представления сценария
| Действие Пользователя             | Отклик Системы                      | 
| --------------------------------- |:-----------------------------------:|
| Нажимает на иконку **Авторизация**| Отображает страницу входа           |
| Вводит логин и пароль             | Отображает введенные значения       |
| Нажимает на кнопку **Войти**      | Отображает страницу личного кабинета|
